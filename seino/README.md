# セイノースタッフサービス - 人材紹介事業 デモサイト（v2：キックオフ改修版）

## 概要

株式会社セイノースタッフサービスの「人材紹介（エージェント）事業」向けのデモ求人ポータルサイトです。

**会社サイト:** https://www.seino-ss.jp/

### v2（キックオフ改修版）について

v1デモで橋戸さんから承認をいただいた5ページ構成をベースに、キックオフで合意形成するための追加3要素を実装したバージョンです。

追加内容：
1. **「セイノースタッフサービスとは」** — about.html の基本情報の前に会社紹介イントロを追加
2. **「個人（求職者）・法人（企業）それぞれへの説明」** — about.html に「ご利用の方へ」2カラムセクションを追加（事業内容の直後）
3. **「支援実績・お客様の声」** — voices.html を新規作成（数値実績4指標＋転職成功者の声3件）。index.html にバナーも追加

## ヒアリング情報

- **案件名:** 人材紹介（エージェント）事業用 求人ポータルサイト新規構築
- **背景:** 既存のジョブギアではSEOが弱く、検索流入が見込めない。また、デザイン面でのブランディングも不十分
- **目的:** エージェント事業の認知拡大と、求職者（正社員希望者）の集客
- **予算:** 約40万円（制作）、月額2万〜3万円（保守・運用）
- **納期:** 5月中までの公開理想
- **重要な要件:** ATS「トルー（Thor）」との連携

## ページ構成

| ページ | 説明 |
|--------|------|
| `index.html` | トップページ。サービス紹介・強みに加え、支援実績バナー（4指標）を掲載 |
| `service.html` | 求人一覧・検索ページ。5件のサンプル求人を掲載 |
| `flow.html` | ご利用の流れ。5ステップで内定までのプロセスを説明 |
| `voices.html` | 支援実績・お客様の声（v2追加）。満足度96%等の4指標＋転職成功者の声3件 |
| `about.html` | 会社概要（v2拡充）。会社紹介イントロ・求職者と企業それぞれへの説明を追加 |
| `contact.html` | 無料相談・お問い合わせページ。FAQも掲載 |

## ATS（Thor）連携準備

現在のデモ段階では、以下の箇所でATS連携を想定した設計を行っています：

1. **service.html の検索フォーム**
   - 職種、勤務地、年収で検索可能な構造
   - バックエンド連携で動的フィルタリング実装予定

2. **求人カード**
   - 各求人にはユニークなIDを付与可能な構造
   - データ属性で拡張可能

3. **セキュリティ**
   - Thor のセキュリティプラグインを考慮した設計
   - 個人情報の厳重な管理体制
